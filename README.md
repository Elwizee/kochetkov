# kochetkov

Решения задач по курсу «Современные технологии и языки программирования» (информационная безопасность).

## Структура

- `Модуль 1/` — задачи 1–4
- `Модуль 2/` — задачи 5–8
- `Модуль 3/` — задачи 9–12
- `Модуль 4/` — задачи 13–16
- `Модуль 5/` — задачи 17–20
- `Модуль 6/` — задача 21
- `data/` — тестовые данные для запуска

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python "Модуль 1/1.py"
python "Модуль 4/13.py"   # Flask, порт 5013
python "Модуль 4/14.py"   # Flask, порт 5014
python "Модуль 4/15.py"   # Flask, порт 5015
```
